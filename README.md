A modern, responsive e-commerce fashion store built with HTML, Tailwind CSS, and JavaScript. FashionHub lets users explore products, add items to the cart, and enjoy a smooth shopping experience. Perfect for showcasing front-end skills and design mastery.

🚀 Features

Fully responsive design for mobile, tablet, and desktop.

Product catalog with images, prices, and descriptions.

Product details page with smooth interactions.

Shopping cart functionality (add/remove/manage items).

Clean UI & animations for a polished look.

Easy to customize & extend for new products or features.

🛠 Tech Stack

HTML5 & CSS3

JavaScript (for interactivity & cart payment getway  logic)

Tailwind CSS (for modern, responsive styling)
